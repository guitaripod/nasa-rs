Quick Start

Install CLI

# Install from source
cargo install --path . --features cli

# Or download pre-built binary
curl -L https://github.com/marcusziade/nasa-rs/releases/latest/download/nasa-cli-$(uname -s)-$(uname -m) -o nasa-cli
chmod +x nasa-cli

Configure & Run

# Set your NASA API key
nasa-cli config set-key YOUR_API_KEY

# Get today's APOD
nasa-cli apod today
